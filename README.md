## 1. Estructura del Proyecto
A continuación se detallan las carpetas principales del repositorio:
1. **src**: Contiene el código fuente de los programas (ej. `primes.py`).
2. **doc**: Documentación técnica y manuales.
3. **bin**: Archivos ejecutables o binarios.
4. **script**: Scripts de automatización o configuración.

---

## 2. Requisitos de Software
Para ejecutar los proyectos de este repositorio, se requiere:
* **Lenguaje de Programación:**
    * Python 3.10 o superior.
    * Administrador de paquetes pip3.
* **Control de Versiones:**
    * Git (configurado con cuenta de GitHub).
* **Entorno de Desarrollo:**
    * VS Code o similar.
